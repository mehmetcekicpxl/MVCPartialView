#  MVC Partial View en ViewComponent Voorbeeld

Dit is een eenvoudige ASP.NET Core MVC project , 
om te laten zien hoe Partial Views en ViewComponents werken.

## Functies

- **Partial View**: Herbruikbare productkaartjes (`_ProductCard.cshtml`)
- **ViewComponent**: Winkelwagen overzicht (`CartSummaryViewComponent`)
 
## Installatie en gebruik

1. Clone de repository
2. Open het project in Visual Studio
3. Build & Run (`F5`) 
4. Bekijk de Productenlijst met Partial View en ViewComponent
